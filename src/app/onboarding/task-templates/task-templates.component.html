About Task Templates

What are Task Templates?

Task Templates allow you to create predefined sets of tasks that can be automatically assigned to new employees during the onboarding process. Instead of manually creating and assigning tasks for each new hire, you can set up templates that handle it all for you.

Key Features

  • Auto-Assignment: Tasks are automatically assigned to employees based on Department, Location, or other filters.
  • Reusable Templates: Create a template once and reuse it for every new onboarding.
  • Applies-To Filters: Use filters like Department, Location, Job Title, etc. to target the right employees.
  • Reduced Manual Work: Eliminate the need to manually add task lists and assign them to onboarding groups.

How It Works

  1. Create a new task template using the Add Task button.
  2. Define the tasks and set the "Applies To" filters (e.g., Department = Engineering).
  3. When a new employee matching the filter is onboarded, the tasks are automatically assigned.
